Tech stack: Django,sqlite3, Django Rest Framework


1. At first copy the repository from Github with SSH KEY with git@github.com:Md-Ibrahim-Khalil/food_delivery.git


2. Create a virtual environment and activate it.

```bash
python3 -m venv "venv"
source venv/bin/activate
```


3. Upgrade to specific pip version for this project.
```bash
pip install --upgrade pip==23.2
```

 Install the required packages.
```bash
pip install -r requirements.txt
